# Task 7 - NLP Basics

## Objective
Perform sentiment analysis using NLP techniques and Naive Bayes.

## Dataset
Custom feedback dataset with 50 records.

## Columns
- Feedback_ID
- Client_Name
- Service
- Feedback_Text
- Rating
- Sentiment

## NLP Tasks
- Lowercasing
- Remove punctuation
- Remove stopwords
- Tokenization
- Stemming
- TF-IDF Vectorization

## Machine Learning
- Naive Bayes Classifier

## Visualizations
- Sentiment Distribution Chart
- Word Cloud
- Confusion Matrix

## Output Files
- coretech_feedback.csv
- sentiment_predictions.csv
- sentiment_distribution.png
- wordcloud.png
- confusion_matrix.png

## Result
Successfully classified positive and negative feedback using NLP and Naive Bayes.
