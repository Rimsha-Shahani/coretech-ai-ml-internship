# Task 08: Neural Network Basics with Keras

## Objective
Build an Artificial Neural Network (ANN) using TensorFlow/Keras and compare its performance with a traditional Machine Learning model.

## Steps Performed

1. Loaded dataset
2. Preprocessed data
3. Encoded categorical values
4. Scaled features
5. Split dataset into train and test sets
6. Built ANN using Sequential API
7. Added two hidden layers with ReLU activation
8. Used Sigmoid activation in output layer
9. Compiled model using Adam optimizer
10. Trained model with validation data
11. Plotted accuracy and loss curves
12. Evaluated model on test data
13. Compared ANN with Random Forest
